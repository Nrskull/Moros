# 故事时间轴页需求


## 需求陈列（已经实现的可以注释掉）


## 当前实现备注

- 展示模式和编辑模式当前共用同一套固定轨道布局，区别只在于是否开放轨道配置入口。
- 顶部世界观 Hero 已改为纯左侧信息布局，不再显示右侧统计块；左侧毛玻璃更强，右侧保留更多封面可见度。
- 轨道背景会在两种模式下持续显示淡色条带，便于稳定识别轨道位置。
- 轨道最左侧已预留一段不映射任何时间的前导空白，时间刻度和事件一起整体右移，避免轨道名被第一张卡片遮挡。
- 标签筛选当前作用于当前世界观范围内的事件集合，并已改为收纳在“固定轨道”行右侧的折叠面板。
- 轨道颜色当前已在两种模式下共同作用于轨道背景识别，卡片纵向位置在缩放和模式切换时保持稳定。
- 编辑模式下，拖动卡片主体会同时支持横向改期与纵向跨轨；拖动卡片右侧手柄会调整结束时间。
- 编辑模式下允许删除非首条轨道，删除后该轨事件会顺延到上一轨末尾。
- 展示模式下，卡片点击用于展开阅读；“记录详情”入口只在展开态显示。
- 展开态卡片已去掉“编辑”按钮，仅保留“编辑详情”和删除操作；轨道信息也已改为显示实际轨道名称而不是数字序号。
- 展示模式下禁用事件拖拽；编辑模式下拖拽结束会立即清理激活态与拖拽态。
- 编辑模式下如果把事件拖到与同世界观、同轨道的其它事件发生重叠，松手后会自动回退到拖拽前的位置。
- 时间轴区域当前整体已关闭文本选择，避免拖拽或点击时出现蓝色选中文本。
