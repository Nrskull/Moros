## 当前实现备注

- 日志模块主入口已改成 `log展示页`，不再把上传、编辑、播放全部堆在同一个页面。
- 展示页按世界观生成目录卡片；每张卡片沿用全局世界观 Hero 的玻璃材质、封面图与主题色联动，但高度压缩为更适合列表浏览的矮比例版本。
- 世界观目录现已改为单列手风琴结构，每个世界观独占一行，避免横向并列后同时展开时难以阅读。
- 点击世界观卡片会展开该世界观下的运行时 log 列表，单条记录支持 `删除 / 编辑 / 打开播放页`。
- 每个世界观展开区最底部固定保留“新增上传 log”行；上传前必须填写“主要角色名”，系统会自动生成 `世界观 - 主要角色名 - 编号` 的标题。
- 日志展示页当前默认会把 `public/logs/test/sealdice-standard-log.json` 作为一条示例记录载入到当前世界观，便于本地测试展示、编辑和播放流转。
- 播放页已支持正文分页：当单条正文超过底部字幕区域时，会在保持角色名固定的前提下自动翻页继续播放，并在播放块上下同时提供“上一页正文 / 下一页正文”按钮。
- 播放页顶部工具条已改成完整横排，倍速控件也已收进同一行，不再因为标题区挤压而竖向堆叠。
