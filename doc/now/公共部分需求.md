# 公共部分需求

## 需求陈列
- 新建世界观的功能应该在切换世界观的下拉栏的最下方，自定义内容有：名字、简介、标签、配图(不必须)
- 同一世界观下，故事时间轴页和年龄编年页，顶部卡片展示的世界观文字保持一致

## 当前实现备注

- 故事时间轴页和角色年龄编年页共用同一套世界观主题变量，因此切页后顶部封面和整体色调保持连续。
- 当前世界观的名称、简介、标签已经移动到 `src/content/worldviews.ts`，顶部下拉菜单也从该配置生成。
- 新增世界观当前通过时间轴页右上方的“新增世界观”抽屉完成，保存后会立即加入顶部世界观下拉菜单，并同步用于时间轴与年龄编年页面。
- 当前封面主题通过独立配置文件按世界观名称映射，后续可继续扩展为每个世界观单独配置封面、说明文案和主题色。
- 未配置封面的世界观会自动切换到 plain Hero 变体，使用更浅的玻璃层与深色文字。
