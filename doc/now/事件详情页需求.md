# 事件详情页需求


## 需求陈列（已经实现的可以注释掉）


## 当前实现备注

- 当前详情页支持编辑标题、摘要、标签、插图地址和正文。
- 支持 `Cmd/Ctrl + S` 快捷保存。
- 当前正文富文本以独立的 `detailHtml` 富文本内容承接，时间轴卡片仍沿用纯文本正文摘录，避免 HTML 直接泄漏到时间轴视图。
- 当前插图采用独立图片区块方案，优先满足“图片区不压书写横线”的阅读要求。
- 当前格式工具条固定停留在正文框顶部，不再跟随选区漂浮。
- 当前格式工具条已增加正文颜色预设与两段式格式刷：第一次点击采样当前位置格式，第二次点击将其应用到当前选区。
- 当前信纸横线已改为由统一 `line-height` 驱动的等距背景线，避免再和文本发生错位或重叠。
- 当前工具栏已改为铺满正文宽度的扁平条带，不再使用独立圆角卡片和阴影。
