# Morosonder

本仓库当前实现的是一个基于 `Svelte + TypeScript + Vite` 的跑团功能网站，站点名为 `Morosonder`。

## 当前范围

当前已实现：

- Mock 时间轴事件数据

- 时间到像素的比例尺映射
- 默认轨道归属的初始化推导
- 绝对定位的横向时间轴渲染
- 世界观一级切换
- 世界观名称、简介、标签已抽到 `src/content/worldviews.ts`
- 世界观封面 Hero 与整页主题联动
- 世界观主题配置已独立抽离
- 顶部页面切换旁提供世界观下拉菜单，并使用毛玻璃材质
- 原“全局设置”已替换为“新增世界观”，保存后会直接加入顶部世界观切换
- 世界观切换使用重叠层海雾过渡，避免白屏闪动
- 世界观 Hero 切换已加入卡片式层叠滑入效果
- 故事时间轴页与角色年龄页顶部 Hero 已改为左侧信息布局，去掉右侧统计块
- `莫名其妙` 世界观已切换为无封面、白色 / 浅灰主导主题

- 新建事件与新增世界观入口
- 抽屉式事件编辑
- 卡片内联编辑 / 删除

- 展示模式 / 编辑模式共用固定轨道布局
- 默认四条轨道已命名为“公共线1 / 公共线2 / 其它线1 / ho404线”
- 时间轴轨道左侧预留了不对应时间的安全区，避免轨道名被首张卡片遮挡
- 编辑模式下的轨道新增、删除与轨道颜色管理
- 新建 / 编辑事件时可指定所属轨道
- 编辑模式下支持跨轨拖拽、横向改期和右侧手柄调节时长
- 编辑模式下若拖拽结果与同轨事件重叠，会自动回退到拖拽前位置
- 标签多选筛选，可在高亮命中与隐藏未命中之间切换，并折叠在固定轨道行右侧
- 固定轨道跨度事件按真实时间宽度贴轨渲染，首尾相接时无缝贴合
- 展示模式下可从展开卡片进入独立的事件详情页
- 展开卡片已移除“编辑”按钮，详情入口统一更名为“编辑详情”
- 时间轴区域禁用文本选择，避免拖拽时出现蓝色选中态
- 事件详情页已接入 `Tiptap` 富文本正文、固定格式工具条、文字颜色、格式刷与页面级保存按钮
- 事件详情页工具栏已改为铺满正文宽度的扁平条带，信纸横线与正文行高保持等距对齐

- 顶部时间刻度自适应抽稀，避免缩小时标签碰撞
- 拖拽漫游与近刻度吸附
- 独立的角色年龄编年页面，可编辑编年节点与角色基准年龄，并自动计算各角色年龄
- 年龄编年页右侧角色图例支持拖拽排序、点击隐藏与删除

- 新增聊天室模块，包含顶部 `聊天室` 入口、公共聊天室、昵称进入层、消息历史、在线成员、新消息提示，以及 `Node.js + WebSocket + SQLite` 的聊天服务骨架

设计说明与 PRD 约束记录在 [doc/README.md](doc/README.md)。

## 命令

```bash
npm install
npm run dev
npm run chat
npm run lan
npm run check
npm run build
```

- `npm run chat`：启动本地聊天服务，默认监听 `127.0.0.1:3031`
- `npm run lan`：同时启动前端开发服务器和局域网可访问的聊天服务，适合在同一 Wi‑Fi 下跨设备测试聊天室
