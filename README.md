# Timeline

本仓库当前实现的是一个基于 `Svelte + TypeScript + Vite` 的本地优先各种跑团时间轴原型。

## 当前范围

当前已实现：

- Mock 时间轴事件数据
- 时间轴事件的本地可编辑状态
- 时间到像素的比例尺映射
- 默认轨道归属的初始化推导
- 绝对定位的横向时间轴渲染
- 世界观一级切换
- 世界观封面 Hero 与整页主题联动
- 世界观主题配置已独立抽离
- 故事时间轴页与角色年龄页顶部 Hero 已改为左侧信息布局，去掉右侧统计块
- `莫名其妙调查` 世界观已切换为无封面、白色 / 浅灰主导主题
- 卡片悬浮聚焦与原位展开
- 新建事件与全局设置入口
- 抽屉式事件编辑
- 卡片内联编辑 / 删除
- 编辑模式下的卡片水平拖拽改期
- 展示模式 / 编辑模式切换
- 展示模式 / 编辑模式共用固定轨道布局
- 编辑模式下的轨道新增、删除与轨道颜色管理
- 新建 / 编辑事件时可指定所属轨道
- 编辑模式下支持跨轨拖拽、横向改期和右侧手柄调节时长
- 标签多选筛选，可在高亮命中与隐藏未命中之间切换，并以右侧悬浮窗呈现
- 固定轨道跨度事件按真实时间宽度贴轨渲染，首尾相接时无缝贴合
- 展示模式下可从展开卡片进入独立的事件详情页
- 事件详情页已接入 `Tiptap` 富文本正文、Bubble Menu 与页面级保存按钮
- 语义缩放：极小视图下通过条件渲染隐藏详细时间文本与摘要，仅保留单行截断标题
- 顶部时间刻度自适应抽稀，避免缩小时标签碰撞
- 拖拽漫游与近刻度吸附
- 独立的角色年龄编年页面，可编辑编年节点与角色基准年龄，并自动计算各角色年龄
- 年龄编年页右侧角色图例支持拖拽排序、点击隐藏与删除

设计说明与 PRD 约束记录在 [doc/README.md](doc/README.md)。
